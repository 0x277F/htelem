template<static_string At>
struct ht::attribute_spec< At, event_handler_attr_type >

Explicit specialization for event handler attributes.

Attributes in an element definition of the form _onclick = [](auto&& event){ } will be converted to children of the element with the type ht::filtered_event_receiver<"onclick", LAMBDA>.
