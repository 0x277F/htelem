Detailed Description

template<static_string Name, class T>
class ht::attribute< Name, T >

A named attribute which can be supplied to an element.

The underlying value type T must be default-constructible and copy-constructible. Internally, when initializing an element, a variable of type T is first default-initialized inside the attribute_list, then replaced by another variable which is copy-initialized during aspect-initialization.

Template Parameters
Namethe attribute name, e.g. "href" for a link or "onclick" for an event handler.
Tthe underlying value type.
