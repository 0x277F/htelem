template<static_string In, class Parent, auto... A>
struct ht::interface_spec< In, Parent, A >

An element's interface specification.

An interface specification consists of an interface name, zero or more parent interface specifications, and zero or more attributes. Every element must have an interface specification, but multiple elements may use the same interface specification – for example, simple text elements like <b> and <i> use the HTMLElement interface. An element which accepts user-defined attributes should provide an alias to a new specialization of interface_spec.

Template Parameters
Inthe interface name, e.g. "HTMLDivElement"
Parentan std::tuple of parent interface_specs
