template<static_string At, class... T>
struct ht::attribute_spec< At, T >

An attribute specification.

Static global attribute_specs are used to initialize attributes inside an element during aspect-initialization. Each attribute_spec has a name and a list of permissible types. For example, the HTML attribute title is implemented as:

inline constexpr attribute_spec<"title", std::string_view> _title;
